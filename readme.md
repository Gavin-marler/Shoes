# helloWorld
## Overview
This program allows the User to input their three favorite shoes and the prices of each, displays them, and displays the total.
## Development Environment
* VS Code
* Java

## Execution
to compile this promgram put:
```java
javac Shoes.java
```
To run put: 
```java
java Shoes
```
